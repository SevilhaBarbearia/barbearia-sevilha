# Roadmap de evolução

## MVP

- Google Login
- Perfil com telefone obrigatório
- Serviços
- Barbeiros
- Horários disponíveis
- Reserva online
- Painel admin
- Pagamento presencial
- Faturamento básico

## Fase 2

- CRUD visual completo para serviços/barbeiros/horários
- Bloqueio de horários pelo painel
- Status de reserva com botões no admin
- Perfil de barbeiro
- Confirmação manual/automática

## Fase 3

- WhatsApp transacional
- Lembretes automáticos
- No-show com histórico
- Relatórios por período
- Comissão por barbeiro

## Fase 4

- Múltiplas unidades
- Programa de fidelidade
- Aplicação SaaS para várias barbearias
- Auditoria avançada
